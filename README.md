# Alert Triage Workflow – Product Design Assignment

This project presents a proposed workflow for investigating and resolving cloud security alerts.

## Problem Statement
Cloud environments generate a high volume of security alerts. Security engineers often need to switch between multiple tools to investigate these alerts, which increases response time and creates inefficiencies.

This design proposes a simplified alert triage workflow to help engineers quickly investigate and resolve security alerts.
